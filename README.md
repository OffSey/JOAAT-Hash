# JOAAT Hash Converter – FiveM

A simple, clean, single-file web tool to convert any string to its **JOAAT hash** as used internally by **FiveM / GTA V**.

## Features

- Converts any string to **Signed (Int32)**, **Unsigned (UInt32)** and **Hexadecimal** hash formats
- Automatically lowercases the input, matching FiveM's internal behavior
- One-click copy for each hash value
- No dependencies – just open `index.html` in your browser

## Usage

1. Clone or download this repository
2. Open `index.html` in any modern browser
3. Type your string and get the hashes instantly

or:

1. Go to: https://offsey.github.io/JOAAT-Hash/

## Preview

| Format | Example (`adder`) |
|---|---|

| Unsigned (UInt32) | `2665046791` |
| Hexadecimal | `0x9F29ABCF` |

## What is JOAAT?

**Jenkins One-at-a-Time (JOAAT)** is a hash function used by Rockstar Games in GTA V and FiveM to identify game entities such as vehicle models, weapon names, peds, and more.
